VSS-Simulator
=============

[![Build Status](https://travis-ci.org/SIRLab/VSS-Simulator.svg?branch=master)][travis]

Mais informações podem ser encontradas em [VSS-SDK][vss-sdk].

Installation
------------
[INSTALL][install]

Cloning
-------
```
$ git clone --recursive https://github.com/SIRLab/VSS-Simulator.git
```

Development and Use
-------------------
```
$ make
$ make proto
$ make run
```
